Text2sound Plugin Tool

[Github] Develop a DAW plugin that generates sound effects from textual descriptions using AI.

Challenge How do I design a text-to-sound tool that maximizes useful output while maintaining a lightweight structure?
My Role Data Collection, Preprocessing, Model Selection, Plugin Development
Year 2024 to Present
Timeline 3 months
Tools Python, Freesound API, PyTorch, Audio Libraries
[visuals]

Project Details

Objective

Develop a digital audio workstation (DAW) plugin with an integrated lightweight text-to-sound model for sound designers to generate and manipulate audio using text prompts.

Frontend

●
User Interface: Designed for seamless DAW integration, built with JUCE.
●
Controls: Text input for sound generation, parameter adjustments for fine-tuning, and real-time playback support.

Backend

●
Generative Model: Utilizes a GAN-based approach on spectrograms to synthesize high-quality audio from text descriptions.

Storage

Local storage for user presets and recent sound generations.

Data Collection

●
Dataset: Curated 5,000+ high-quality .wav files from Freesound API, covering percussive sound categories (e.g., kicks, snares, ambience, synths).
●
Metadata: A structured CSV file containing descriptive attributes for training and fine-tuning the model.
