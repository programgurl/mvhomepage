Leetcode & Interview Prep Guide

I am writing this study guide as a possible resource to others doing interview prep and to track my own learning progress.

Below I detail a step-by-by guide for acing the technical portion of SWE interviews (in Python).

The Fundamentals

Programming Languages:

Data Structures

Arrays

Here is a table with core methods for this data structure:

Method Description Example Time Complexity
append(value) Adds an item to the end of the list. my_list.append(5) O(1)
extend(iterable) Adds all items from an iterable to the list. my_list.extend([1, 2, 3]) O(k)
insert(index, value) Inserts an item at a specified position. my_list.insert(1, "new") O(n)
remove(value) Removes the first occurrence of a value. my_list.remove(2) O(n)
pop(index=-1) Removes and returns the item at the specified position (default: last). my_list.pop() O(1) (last), O(n) (arbitrary index)
clear() Removes all items from the list. my_list.clear() O(n)
index(value, start=0, stop=None) Returns the index of the first occurrence of a value. my_list.index(3) O(n)
count(value) Counts the occurrences of a value in the list. my_list.count(2) O(n)
sort(key=None, reverse=False) Sorts the list in ascending order (or by key). my_list.sort() O(n log n)
reverse() Reverses the elements of the list in place. my_list.reverse() O(n)

I find the sort function with the use of the key to be the most useful of all the methods. So let's go over an example in detail.

my_list.sort(key= lambda x: x)
