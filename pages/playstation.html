PlayStation Facial Texture Generator

Trained a generative AI model on a large-scale (50k+ image) facial UV-texture dataset to generate brand new face textures in under 10 seconds as a tool for animators.

Challenge What generative tool could animators use to speed up the animation pipeline in regards to UV facial textures and how do you build it?
My Role Research, Data Collection, Model Training, UI Development
Year 2023
Timeline 3 months
Tools Python, PyQt, AWS, Jira
[visuals]

Project Details

Objective:

Add a feature to an internal tool which allows artists to generate new face textures. Users should be able to edit a series of features in the image such as skin tone, age, facial hair, etc.

Context:

GANs or Generative Adversarial Networks have widely been used for image synthesis but fail to offer any control over the generated image. GANSpace uses Principal Component Analysis to find series of features that vary within the training data.

Dataset:

Open-source over 50k UV facial texture samples.
